Link formats

  • Relative link
  • Relative link: rel="external"
  • Relative link: rel="external", target="foo"
  • Relative link: data-rel="dialog" (not tracked in history)
  • Absolute links
  • Absolute link
  • Absolute link: rel="external"
  • Absolute link: rel="external", target="foo"
  • Email links
  • Basic email: mailto:jdoe@foo.com
  • Mailto with a cc: and bcc:
  • Mailto with a subject and body pre-filled
  • Phone links
  • Phone: tel:15555555555
  • Phone: wtai://wp/mc;15555555555
  • Phone: dc:234*234*234
  • Other
